RITA SORRY WEBSITE

1. Open index.html in Chrome to preview.
2. To add background music, put an MP3 file in the same folder and rename it to: our-song.mp3
3. Upload index.html and our-song.mp3 to a GitHub repository.
4. In GitHub: Settings > Pages > Deploy from branch > main / root > Save.

The website is fully mobile-responsive and uses no external library.
